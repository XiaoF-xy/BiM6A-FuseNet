# BiM6A-FuseNet v1b Projected Concat Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `v1b_proj256_concat`, which projects the BiRNA and handcrafted branches independently to 256 dimensions before concatenation while preserving the complete v1 evaluation protocol.

**Architecture:** Keep all hash-protected v1 feature-extraction modules unchanged. Add a focused module containing a reusable projected-concat head and a v1b classifier that reuses the existing FiLM feature builder and handcrafted CNN. Select it through one explicit config/CLI flag so v1 remains the default.

**Tech Stack:** Python 3.10, PyTorch, Transformers, PEFT, pytest/unittest, existing BiM6A-FuseNet training pipeline.
